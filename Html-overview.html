What is an HTML?
HTML stands for Hyper Text Markup Language.
The HyperText Markup Language, or HTML is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets and scripting languages such as JavaScript.

Hypertext: A text which contains links to other texts.

Hypermedia: It is a broader term for the audio files, text files and videos over the internet.

Markup: It means to mark something and add notes.

Language: HTML is a coding language. The use of the term language implies that it has a syntax. This means that there is a right and a wrong way to express or code it.

Parsing: THe process of the browser reading and trying to make sense of your HTML code is called Parsing.

HTML : Elements and Tags

HTML Element:
An HTML element is a type of HTML document component, one of several types of HTML nodes. HTML document is composed of a tree of simple HTML nodes, such as text nodes, and HTML elements, which add semantics and formatting to parts of document. Each element can have HTML attributes specified.

HTML-Tags:
HTML tags are like keywords which defines that how web browser will format and display the content. With the help of tags, a web browser can distinguish between an HTML content and a simple content. HTML tags contain three main parts: opening tag, content and closing tag. ... Each HTML tags have different properties.

Syntax for any HTML Tag:
 A tag is composed with special characters: < , > and /. They are interpreted by softwares to compose an HTML element.
An HTML tag is a special word or letter surrounded by angle brackets, < and >. You use tags to create HTML elements , such as paragraphs or links. HTML-Element Attributes:
HTML attributes provide additional information about HTML elements.

HTML: Attributes
